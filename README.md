# 🎵 VIBE - Ultra Premium Music Player

VIBE is a high-performance, visually stunning music streaming platform built with the **MERN Stack** (MongoDB, Express.js, React, Node.js). It features a modern, luxury UI inspired by Spotify and Apple Music, with a focus on smooth animations, glassmorphism, and premium aesthetics.

![VIBE Preview](https://images.unsplash.com/photo-1614613535308-eb5fbd3d2c17?w=1200&h=400&fit=crop)

## ✨ Features

- **💎 Ultra Premium UI**: Deep charcoal theme with neon purple and gold accents.
- **📱 Fully Responsive**: Seamless experience on Desktop, Tablet, and Mobile.
- **📻 Advanced Player**: Sticky bottom player with seek control, volume, and playback management.
- **🔐 Secure Auth**: JWT-based authentication for user profiles and libraries.
- **🚀 PWA Support**: Installable as a native app on iOS, Android, and Desktop with custom high-fidelity icons.
- **⚡ Fast Performance**: Optimized React components and fast API responses.
- **📂 Library Management**: Create playlists, save favorites, and browse trending tracks.

## 🛠️ Tech Stack

- **Frontend**: React, Vite, Tailwind CSS (v4), Framer Motion, Lucide React.
- **Backend**: Node.js, Express.js.
- **Database**: MongoDB (Mongoose).
- **Storage**: Cloudinary (for Audio & Images).
- **Other**: JWT, Multer, Axios.

## 🚀 Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (v18+)
- [MongoDB](https://www.mongodb.com/try/download/community) account or local instance
- [Cloudinary](https://cloudinary.com/) account for storage

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/music-vibe.git
   cd music-vibe
   ```

2. **Backend Setup**
   ```bash
   cd server
   npm install
   # Create a .env file and add your credentials (see .env template)
   npm run dev
   ```

3. **Frontend Setup**
   ```bash
   cd ../client
   npm install
   npm run dev
   ```

## 📱 PWA Installation
Navigate to `http://localhost:5173` on your mobile browser or desktop. Look for the **"INSTALL VIBE APP"** button on the Home or Login page to install it as a native application.
